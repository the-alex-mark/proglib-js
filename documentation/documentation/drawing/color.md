# Color
Предоставляет методы и свойства для работы с цветовыми моделями.

## Инициализация
```js
const Drawing = require('proglib').Drawing;
const Color = Drawing.Color;

// Создает объект Color из четырех значений компонентов RGBA (Красный, Зеленый, Синий и Прозрачность).
// 
// Параметры:
//    Red
//       Красный: { 0 ... 255 }.
//    Green
//       Зелёный: { 0 ... 255 }.
//    Blue
//       Синий: { 0 ... 255 }.
//    [Alpha]
//       Прозрачность: { 0 ... 1 }.
// 
// Возврат
//    Color
let _color_ = Color.FromRgba(Red, Green, Blue, Alpha);

// Создает объект Color из указанного имени предопределенного цвета.
// 
// Параметры:
//    Name
//       Name Строка, которая является именем предопределенного цвета.
// 
// Возврат
//    Color
let _color_ = Color.FromName(Name);
```

## Свойства
```js
// Возвращает имя этого аспекта Color.
// 
// Возврат:
//    String
_color.Name;

// Получает значение красного компонента этого объекта Color.
// 
// Возврат:
//    Number
_color.R;

// Получает значение зеленого компонента этого объекта Color.
// 
// Возврат:
//    Number
_color.G;

// Получает значение синего компонента этого объекта Color.
// 
// Возврат:
//    Number
_color.B;

// Получает значение альфа-компонента этого объекта Color.
// 
// Возврат:
//    Number
_color.A;

// Получает значение, указывающее, является ли объект Color именованным цветом.
// 
// Возврат:
//    Boolean
_color.IsNamedColor;

// Возвращает значение, показывающее, является ли объект Color системным цветом.
// Системным является цвет, который используется в элементе отображения Windows.
// 
// Возврат:
//    Boolean
_color.IsSystemColor;
```
