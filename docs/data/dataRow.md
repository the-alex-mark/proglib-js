# `DataRow`
Представляет строку данных в [`DataTable`](./dataTable.md).

## Инициализация
```js
const Data = require('proglib').Data;
const DataTable = Data.DataRow;

// Инициализирует новый экземпляр объекта `DataRow` с указанной строкой.
// 
// Параметры:
//    Values
//       Массив значений строки различного типа.
let _row = new DataRow(...Values);
```

## Свойства
```js
// Возвращает данные объекта DataRow в массиве.
// 
// Возврат:
//    Number
_row.Data;

// Возвращает общее число элементов в коллекции.
// 
// Возврат:
//    any[]
_row.Count;
```

## Методы
```js
// Возвращает значение, позволяющее определить, представляют ли текущий объект DataRow и указанный объект `DataRow` одно и то же значение.
// 
// Параметры:
//    Value
//       Объект DataRow, сравниваемый с текущим объектом DataRow.
// 
// Возврат:
//    Boolean
_row.Equals(Value);

// Возвращает тип для текущего экземпляра.
// 
// Возврат:
//    String
_row.GetType();
```
