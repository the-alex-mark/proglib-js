# Size
Сохраняет упорядоченную пару целых чисел, указывающих `Height` и `Width`.

## Инициализация
```js
const Drawing = require('proglib').Drawing;
const Size = Drawing.Size;
const Point = Drawing.Point;

// Инициализирует новый экземпляр структуры Size из указанного объекта Point.
//
// Параметры:
//    pt
//       Объект Point, из которой инициализируется эта структура Size.
let _size = new Size(pt);

// Инициализирует новый экземпляр структуры Size из указанных измерений.
//
// Параметры:
//    Width
//       Компонент ширины нового Size.
//
//    Height
//       Компонент высоты нового размера Size.
let _size = new Size(Width, Height);
```

## Свойства
```js
// Проверяет, равны ли 0 ширина и высота структуры Size.
// 
// Возврат:
//    Boolean
_size.IsEmpty;
```

## Методы
```js
// Округляет значения текущего объекта Size до ближайших целых чисел.
_size.Round();

// Проверяет, является ли указанный объект с таким же размером, что и объект Size.
// 
// Параметры:
//    sz
//       Объект Size для проверки.
_size.Equals(sz);

// Создает удобную для восприятия строку, представляющий этот объект Size.
// 
// Возврат:
//    String
_size.toString();

// Возвращает тип для текущего экземпляра.
// 
// Возврат:
//    String
_size.GetType();
```

```js
// Получает объект Size, имеющую значения Height и Width, равные 0.
// 
// Возврат:
//    Size
Size.Empty();

// Прибавляет ширину и высоту одной объекта Size к ширине и высоте другой объекта Size.
// 
// Параметры:
//    sz1
//       Первая складываемая структура Size.
//    sz2
//       Вторая складываемая структура Size.
// 
// Возврат:
//    Size
Size.Add(sz1, sz2);

// Округляет значения объекта Size до ближайших целых чисел.
// 
// Параметры:
//    sz
//       Округляемый объект Size.
// 
// Возврат:
//    Size
Size.Round(sz);

// Вычитает ширину и высоту одной объекта Size из ширины и высоты другой объекта Size.
// 
// Параметры:
//    sz1
//       Структура Size, которая находится слева от оператора вычитания.
//    sz2
//       Структура Size, которая находится справа от оператора вычитания.
// 
// Возврат:
//    Size
Size.Subtract(sz1, sz2);
```
